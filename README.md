SSHX 云端终端 · GitHub Actions 一键搭建
 
基于 GitHub Actions 实现的 免费 Ubuntu 22.04 云端终端
自带完整 Android 编译环境、图形桌面转发、网页终端、文件传输功能，零服务器、零配置、一键启动。
 
📌 重要前置（必须操作）
 
使用前必须先 Fork 本仓库！
 
1. 点击仓库右上角  Fork 
2. 默认参数直接创建
3. 必须在你自己 Fork 的仓库运行工作流，否则无法使用
 
🚀 完整使用教程
 
1. 进入你 Fork 的仓库 → 打开  Actions 
2. 选择工作流 → 点击  Run workflow  启动
3. 系统自动初始化环境、安装所有依赖（耗时 5–10 分钟）
4. 等待日志输出 sshx.io 网页链接
5. 复制链接，浏览器打开即可进入云端网页终端
 
💻 首次进入终端说明
 
- 第一次打开不会自动显示终端窗口
- 根据提示设置用户名
- 设置完成后点击页面 「+」加号 即可新建终端正常使用
 
⚙️ 预装环境详情（完整编译环境）
 
 
 
🛠️ 终端卡顿 / 卡死解决办法
 
- 建议使用优质代理网络，否则高延迟极易卡顿
- 输入卡死：左侧窗口「三个点」关闭终端
- 关不掉：直接点 + 新建终端
- 彻底卡死：直接刷新网页即可恢复
 
📂 三种文件传输方式（任选其一）
 
1. Git 推送至自己私有仓库（推荐编译产物）
 
适合上传 IMG 镜像、编译成品
 
 
 
2. VNC + noVNC 桌面穿透传文件
 
可启动完整图形桌面
浏览器进入远程桌面 → 可直接网页上传下载文件、打开浏览器、使用图形浏览器
 
3. 自建 HTTPS 直链传输
 
远程终端使用  wget  /  curl  拉取直链文件，适合快速批量拉取资源
